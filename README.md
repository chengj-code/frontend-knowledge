## 模块说明

| 模块 | 说明 |
|------|------|
| **TypeScript** | 类型系统、泛型、高级类型、工具类型、工程配置 |
| **JavaScript** | ES6+ 语法、原型链、闭包、事件循环、异步编程 |
| **HTML-CSS** | 语义化标签、布局方案、动画、响应式设计 |
| **React** | Hooks、状态管理、虚拟 DOM、服务端渲染 |
| **Vue** | 响应式原理、Composition API、Vuex/Pinia |
| **Nodejs** | 流、Buffer、模块化、Express/Koa |
| **工程化** | Webpack/Vite、Monorepo、Git 工作流 |
| **性能优化** | 加载优化、渲染优化、监控体系 |
| **网络** | HTTP 协议、缓存策略、安全、TCP/IP |
| **浏览器** | 渲染流程、垃圾回收、Web API |

## 各模块内容规范

每个模块文件夹下包含两个子目录：

- **`basics/`** — 系统化的基础知识文档，适合入门学习与查阅
- **`interviews/`** — 分级面试题库（基础/进阶/专家），附参考答案要点
