## 🎯 目标目录格式（参考 HTML-CSS 标准）

```markdown
## 📚 问题速查目录

> **快速导航**：点击题目即可跳转到对应位置 | 共 **XX** 道题（基础X + 进阶X + 专家X）

### 基础层（★☆☆）Q01 - QXX

| 题号 | 题目 | 难度 |
|:----:|------|:----:|
| [Q01](#q01-标题转锚点) | 题目文本 | ★☆☆ |

### 进阶层（★★☆）QXX - QXX
### 专家层（★★★）QXX - QXX
```

**关键特性：**
1. ✅ 三级分层展示（基础/进阶/专家）
2. ✅ 表格化呈现（题号 + 题目 + 难度）
3. ✅ Markdown 锚点链接（点击即跳转）
4. ✅ 统计信息（总题数和各层数量）
5. ✅ 插入位置：文档开头、总览之后、正文之前

## 📝 执行步骤

### 步骤 1：提取各文档的题目信息

对每个文档执行：
1. 使用 Grep 提取所有 `^## Q\d+:` 或类似的题目行
2. 识别题目标题、难度标识（★☆☆ / ★★☆ / ★★★）
3. 确定插入位置（通常在总览表格后、第一道题之前）
4. 生成对应的 Markdown 锚点（将标题转为小写+连字符格式）

### 步骤 2：生成目录内容并插入

对每个文档：
1. 按难度分组整理题目
2. 生成符合标准格式的 Markdown 目录表格
3. 在正确位置插入目录内容
4. 替换或删除旧的简单目录（如有）

### 步骤 3：验证跳转功能

对每个文档：
1. 确认所有锚点链接格式正确
2. 确认题目行的 ID 与锚点匹配
3. 抽查几个链接确认能正确跳转
