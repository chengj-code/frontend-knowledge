## 推荐阅读顺序

```
第一阶段：语言基础
  JavaScript → TypeScript

第二阶段：框架深入（选一个主攻）
  Vue 或 React

第三阶段：工程化与生态
  工程化 → 性能优化 → 网络 → 浏览器 → Node.js

第四阶段：源码级进阶（中高级开发者）
  Vue2 源码解读 → Vue3 源码解读 → React 源码解读
```
